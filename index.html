Korean Novel Chapter Editor

Upload a .txt file to begin. The editor preserves all original formatting and line breaks.

🔧 How to Use:

  1. Upload your Korean novel (click 📁 icon)
  2. Split chapters (click ✚ icon) - now supports MULTIPLE regex patterns!
  3. Add patterns for different heading formats like "Episode 1", "Chapter 4", "7th Episode", "1026회", "189"
  4. Preview the splits before applying
  5. Navigate chapters using ◀ ▶ buttons (now auto-scrolls to top!)
  6. Scroll to top/bottom using the ↑ ↓ buttons (full page scroll)
  7. Edit any chapter - click the ✏️ pen icon on bottom bar
  8. Save edits - changes reflect in full text
  9. Download (💾 icon) and open in Notepad++
